# Decision Records

Lightweight ADRs. One file per finalized decision, numbered.

Format per file:
- **Context** — what forced the decision
- **Decision** — what we chose
- **Alternatives** — what we rejected and why
- **Consequences** — what this locks us into

When a decision in [../considerations.md](../considerations.md) gets finalized, promote it to a numbered doc here and link both ways.
