# Color Smash Support

## Quick help

- **Plugin doesn't appear in Photoshop**: ensure you're on Photoshop
  2024 (25.0.0) or later, then look under `Plugins > Color Smash` or
  `Window > Extensions > Color Smash`.
- **Layer dropdowns look stale**: click the ⟳ button next to the
  document dropdown — it forces a fresh read.
- **Apply produces a wrong color shift**: make sure the source layer
  isn't an adjustment layer (the plugin matches pixel layers and
  smart objects). If you want only color (not luminance), enable
  **Hue only** under Match controls.

## Reporting bugs

Open an issue with as much detail as you can:

**https://github.com/erichgschmidt/colorsmash/issues**

Helpful things to include:
- Photoshop version (`Help > System Info`)
- Operating system
- Steps to reproduce
- A short screen recording if the issue is visual
